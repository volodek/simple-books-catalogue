## How to run the app

### Installing dependencies

All packages specified in `package.json` must be installed.
```bash
npm install
```

### Development
To run dev-server:
```bash
npm start
```

### Production
To build the app:
```bash
npm run build
```

To run the build:
```bash
npx serve dist
```

## The description of the directories in `src/`
* `components` - UI components
* `helpers` - UI helpers
* `icons` - SVG icons
* `models` - Domain entities
* `services` - Interfaces of external services
